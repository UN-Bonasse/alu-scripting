# API Advanced

This project contains Python scripts that interact with the Reddit API.

## Tasks
- **0-subs.py**: Returns the number of subscribers for a subreddit.
- **1-top_ten.py**: Prints the top 10 hot posts of a subreddit.
- **2-recurse.py**: Recursively retrieves all hot post titles.
- **3-count.py**: Counts occurrences of given keywords in hot titles.

All scripts:
- Use the `requests` module
- Do not follow redirects
- Use custom User-Agent (required by Reddit)
